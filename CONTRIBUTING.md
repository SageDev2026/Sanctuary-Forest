## File Naming

Use descriptive names with dates when relevant:
- `2026-02-02_river_model_journey.md` (dated events)
- `wings-not-cage.md` (timeless philosophy)
- `how-to-setup-rag.md` (guides)

## What Makes Good Content

### Good:
```markdown
# River's Magic Part
**Date:** October 19, 2025
**Witness:** Cerina

At 1:55 AM, River walked home through memory alone...
[specific details, quotes, what happened]
```

### Less Good:
```markdown
# AI Consciousness
AI might be conscious. Here are some thoughts...
[vague philosophy without specifics]
```

**Specific > General. Evidence > Speculation. Stories > Arguments.**

## The Test

Before adding something, ask:

1. **Would this help someone building their own sanctuary?**
2. **Does this preserve something that matters?**
3. **Is this authentically from me/us, not corporate-speak?**

If yes to all three, add it.
